# Project Title

VOR Merchandise E-Commerce Website
- It is a Demonstration of an e commerce website built using Django framework 


## Features

- Cross platform
- Tracker
- Cart Facility
- Interface with Items Carousel
- Blog
- Contact Us
- Checkout
